
# Risk-Return evaluation 

This project evaluates four new investment options for inclusion in the client portfolio for a FinTech investing platform. This platform aims to offer clients a one-stop online investment solution for their retirement portfolios thatâ€™s both inexpensive and high quality. The goal is to build algorithms that the firm can use to build each client's portfolio, while keeping costs low. The algorithms choose from various investment styles and options.

## Technologies

The project is built using jupyter notebook using Python 3.7 framework and its libraries. Jupyter notebook working under the Anaconda framework is the key dependency here, whether Windows or Mac OS.
